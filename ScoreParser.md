# ScoreParserの使い方   
ScoreParserは楽譜のようにSequencerの流れを記述するシステムです。  
気が向いたら書きます。  

## ScoreParserをincludeする   
```js
  const {ScoreParser} = fisce.foxUtils;  
```

## クラスの生成  
こんな感じでクラスを生成する。  
```js
  const SP = new ScoreParser();  
```

## シンボルとセパレータ
　シンボルとは実行可能単位である。それらは命令である。具体的には「コンソールにこれこれを出して！」「このオブジェクトを作って！」「この音を鳴らして！」など。セパレータとはシンボルを分割するという概念である。このパース機構の場合、それは大文字のアルファベットになる。何か具体的な記号があるわけではない。すなわち、それぞれのシンボルはただ一つの大文字のアルファベットから始まるので、それ以降に小文字や数字などを連ねる形である。たとえばAppleとかBananaという感じ。使える文字の一覧：
```
  A-Z, a-z, 0-9, ^_+-.
```
なお特殊記号として[]と{}が使えるが、説明は後述。シンボルに使えるのはこれらだけ。さっきも述べたようにただ一つの大文字のアルファベットで始まる。だからたとえば、
```
  CatDog22XYZSea4lionKing
```  
となっている場合、
```
  Cat, Dog22, X, Y, Z, Sea4lion, King
```
と分けられ、それぞれが特定の意味を持つか、無視される。無視されるとしても意味を持つ。その説明も後で。  

## イベントとその種類
　イベントとはシンボルに割り当てられる実行内容である。シンボルの並びは、イベントの実行順や実行間隔を表現する。イベントにはSpotEventとBandEventという二つの種類がある。  
### SpotEvent
　SpotEventはそのタイミングで実行される関数である。その瞬間に実行されて、それっきり。なお内容はリセットにより再利用されるが、定義の仕方によってはその限りではない場合がある。再利用されることを前提として運用されることがほとんどの場合想定される。たとえばコンソール出力、オブジェクトの生成、音の発振など。  
### BandEvent  
　BandEventはそのタイミングで発火し、そのあと一定の時間の間なにかしらを実行するものである。引数にプログレス(0～1)を取ることができる。終了するとイベントは破棄される。これも基本的には、繰り返し実行されることが想定される。  

## イベントをシンボルに登録する
　シンボルにイベントを割り当てるにはScoreParserのインスタンスメソッドを使う。いくつか方法がある。
### addSpotEventSeed, addBandEventSeedを使う
　ダイレクトにシンボルとイベントを結び付ける方法として、これらの関数を使うやり方がある。なおなぜEvent「Seed」と呼ぶかというと、結局発火のタイミングまで含めてイベントなのだが、割り当ての段階では発火タイミング（キー）が分かんないので、種の状態（Seed）というわけ。使い方は簡単で、さっき述べたようにアルファベットの大文字から始まるそれ...と、関数を使う。
```js
  SP.addSpotEventSeed("Action_spot", () => { /* なんか実行する */ });
```
"Action_spot"がシンボル名で、これによりシンボル「Action_spot」に関数が割り当てられ、然るべきタイミングで実行される。また、BandEventの場合はaddBandEventSeedを使う。
```js
  SP.addBandEventSeed("Action_band", () => { /* なんか実行する */ }, {
    duration:someDuration, /* 実行する時間の長さ */
    fire:someFunction, /* 発火時に実行してほしい関数 */
    finish:someFunction /* 終了時に実行してほしい関数 */
  });
```
これで然るべきシンボルに関数が割り当てられる...のだが、いちいちこんなことをしていてはいくつあっても足りないし、たとえば演奏などをさせる場合、音の数だけちまちま用意しないといけなくなってしまう。それは非常に手間である。もちろんforループである程度簡略化もできるが応用が利かないだろう。そこで一般的には、modとlibという仕組みで、一定の規則に従ってシンボルにイベントシードを割り当てる方法が使われる。
